发表于 2019-01-10

搭建步骤

1.注册Git账号

主要有三项，1.用户名 2.邮箱 3.登录密码。

2.创建项目

进入主页找到 new repositories 一个绿色的按钮。
Repository name（仓库名称，写 你的用户名.github.io 因为要用到 Github Pages 即你的域名。）
Initialize this repository with a README 这个选项勾上
租后 create repositories 仓库就创建成功啦

3.安装Git

Windows: 下载安装Git
Mac: 使用 Homebrew（貌似自带有Git）
Linux（Ubuntu，Debian）：sudo apt-get install git-core
Linux（Fedora，Red Hat，CentOS）：sudo yum install git-core

4.配置ssh

第一步、看看是否存在SSH密钥(keys)

第二步、创建一对新的SSH密钥(keys)

第三步、在GitHub账户中添加你的公钥
